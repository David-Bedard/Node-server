# Node server
TCP, HTTP example built with NodeJS

## Prerequisites

Node latest version

## Installing
In a cmd at the project's root:
```
npm install
```

## Start server and/or client
In a cmd at the project's root:
```
npm start
```

## Net (TCP) Documentation
https://nodejs.org/api/net.html

## HTTP noderest framework
